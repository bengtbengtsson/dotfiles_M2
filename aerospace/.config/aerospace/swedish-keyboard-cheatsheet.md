# AeroSpace Swedish Keyboard Cheat Sheet

These bindings differ from US layout due to physical key positions.

| Action              | US key               | SE key                  |
|---------------------|----------------------|-------------------------|
| Layout toggle       | `alt + /`            | `alt + -` (bottom row)  |
| Resize -50          | `alt + -`            | `alt + +` (top row)     |
| Resize +50          | `alt + =`            | `alt + ´`               |
| Fullscreen toggle   | `alt + '`            | `alt + Ä`               |
| Service mode        | `alt + shift + ;`    | `alt + shift + Ö`       |

All other bindings (hjkl, letters, enter, comma, tab) are the same on both layouts.
